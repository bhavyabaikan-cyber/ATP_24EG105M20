
# Week-4 HTML Practice – README


This Week contains two simple HTML web pages created for practicing basic HTML concepts.
The pages demonstrate the usage of:

* Headings
* Paragraphs
* Lists
* Images
* Hyperlinks
* Horizontal lines
* Basic webpage structure

The purpose of this project is to understand the fundamentals of HTML and webpage creation.


---

# Learning Objectives

Through this project, the following HTML concepts are practiced:

* Structure of an HTML document
* Using headings and paragraphs
* Creating ordered and unordered lists
* Adding images to webpages
* Creating hyperlinks
* Using comments in HTML
* Page organization and formatting
